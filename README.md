# BTS Legacy

A modern, interactive fan website built using **HTML, CSS, and JavaScript** to celebrate the journey, music, members, and achievements of BTS. This project focuses on immersive visuals, smooth animations, creative UI interactions, and a rich fan experience.

## FEATURES:

### Home / Intro Section:

- Fullscreen cinematic video background
- Animated hero section
- Dynamic changing text lines
- Smooth navigation bar
- Explore button for quick access

### Members Section:

- Interactive member cards slider
- Coverflow carousel effect using Swiper.js
- Smooth card transitions
- Clickable member profiles

### Member Info Popup:

- Stylish animated popup design
- Member details and achievements
- Unique paper-style open/close animation
- Explore Discography button
- Dynamic content loading with JavaScript

### BTS Group Albums:

- Horizontal scrollable album strip
- Custom album card design
- Realistic CD disc animation
- Hover effects with sliding disc
- Dynamic background changes per album

### Album Popup:

- Book-style opening popup animation
- Display album title and songs
- Click song to preview on Spotify
- Embedded Spotify player
- Close popup with smooth transition

### Solo Discography:

- Separate discography page for all BTS members
- RM, Jin, SUGA, j-hope, Jimin, V, Jungkook sections
- Explore button opens exact member section
- Horizontal album strips for each member
- Same animated album/CD experience

### Timeline Section:

- BTS journey milestones
- Important achievements by year
- Styled timeline layout

### UI / Experience:

- Premium dark theme with purple glow aesthetic
- Responsive layout for desktop and mobile
- Smooth scrolling navigation
- Hover animations and transitions
- Clean modern design

### Additional Features:

- Spotify track preview integration
- Dynamic DOM manipulation
- Reusable popup system
- Organized multi-page project structure

## TECH STACK:

- HTML5
- CSS3
- JavaScript (ES6)
- Swiper.js
- Spotify Embed API
- DOM API
- Responsive Web Design
